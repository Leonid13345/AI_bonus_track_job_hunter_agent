# Job Hunter Agent

**Агент для поиска работы, анализа вакансий и подготовки к собеседованиям**

## Задача, для кого и какая боль сейчас

Соискатели тратят много времени на просмотр десятков вакансий, анализ требований и подготовку откликов. Сложно оценить реальное соответствие резюме вакансии и понять, какие навыки подтянуть. HR-ы тоже тонут в откликах.

## Что именно сделает PoC на демо

Пользователь загружает резюме (PDF/текст) или описывает опыт. Агент:
1. Парсит вакансии с внешних сайтов (HH.ru, LinkedIn) по заданным критериям.
2. Сравнивает требования с профилем пользователя, подсвечивает несоответствия.
3. Предлагает персонализированные советы по улучшению резюме.
4. Генерирует сопроводительные письма под конкретную вакансию.
5. Может подготовить список вопросов для собеседования и потренировать ответы.
6. Отслеживает статусы откликов (пользователь вносит вручную).

## Что НЕ делает PoC (out‑of‑scope)
- Автоматический отклик на вакансии.
- Гарантия трудоустройства.
- Интеграция с платными премиум-функциями.
- Парсинг сайтов, запрещающих автоматизацию (только официальные API).
